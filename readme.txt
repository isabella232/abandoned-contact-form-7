Abandoned Contact Form 7 Pro provides an ability to track the data from Contact Form 7 even if the user does not submit the form.

== Description ==

Zeal Abandoned Forms plugin provides to track the Contact Form 7 form data even if the user does not submit the form. And, sends mail notification to each Abandoned user.
Zeal Abandoned Forms plugin also provides the separate functionality to enable/disable the Abandoned form.
Zeal Abandoned Forms plugin manages form list items.
Zeal Abandoned Forms plugin sends notification mail to each guest user,  if the user does not submit the form.
Zeal Abandoned Forms plugin set time events for mail notifications.
Zeal Abandoned Forms plugin shows other information like IP address and other useful information.
Zeal Abandoned Forms plugin exports the all Abandoned form data which helps to track the visitor information.

= Features of Abandoned Contact Form 7 Pro =

* Select your email ID field for Form in the form settings.
* Enable/Disable Option for multiple forms where you need in Contact Form 7.
* View the list of entries in the Admin side for Abandoned submission.
* In the list of Abandoned forms entry in admin side you can view IP addresses and other data which are filled-up by the user.
* Provision to send an email to a single user that abandoned the form.
* Facility to export the list of entire abandoned users.
* You can schedule time for sending email to each abandoned user.
* Set the Status of the user to send an email so we can manage not send the same email again to the same user.
* Manage the number of time emails sent to the user.
* Also, An option to customize the mail body for Abandoned email.

== Installation ==

Installing the plugin is easy. Just follow these steps:

1. From the dashboard of your site, navigate to Plugins --> Add New.
2. Select the Upload option and hit "Choose File."
3. When the popup appears select the plugin-file-name.zip file from your desktop.
4. Follow the on-screen instructions and wait till the upload completes.
5. When it's finished, activate the plugin via the prompt. A message will display confirming activation was successful.

== Frequently Asked Questions ==

= Can I configure SMTP ? =

Yes, you can choose either SMTP or Default PHP mailer.

= Can I send mail notification to every abandoned user? =

Yes, you can send mail notification to either every abandoned user or a particular user.

= Can I enable/disable mail notification for particular users? =

Yes, you can easily enable/disable mail notification for particular users.

= Can I Abandoned the data of file type input field of the form =

No, you can't Abandoned the data of file type input field.
